baseLayer

features

layers
layers Holds all the layers in the map
{int} scale
Represents the "width" of the baseLayer. A scale of 100 means that the base is 100 arbitrary units long. All other layers are relative to this scale. The height is defined as ImHeight/ImWidth * scale
